TOTP Vault

This application provides time-based one-time passwords (TOTP) for shared service accounts. Access is restricted to authorized personnel only.

How to Access

  1. Navigate to the password entry in IT Glue
  2. Click the TOTP link associated with the account
  3. Authenticate with your corporate credentials
  4. The current TOTP code will be displayed
/code/{password-id}

Direct access requires a valid password ID from IT Glue.
